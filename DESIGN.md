# CHADALU Enterprises - Design System

## Design Principles
- **Mobile-first** - Design for smallest screens first, then scale up
- **Responsive** - Works on all devices: mobile, tablet, laptop, desktop
- **Accessible** - WCAG compliant
- **Fast** - Optimized for Kenya's internet speeds

## Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Primary Gold | `#C8922A` | CTAs, highlights, brand |
| Dark | `#080808` | Background |
| Dark Secondary | `#121212` | Cards, sections |
| Cream | `#F2EBD9` | Primary text |
| Cream Secondary | `#A89F8C` | Secondary text |
| Accent Green | `#2E9E6B` | Success, available |
| Accent Red | `#B83030` | Errors, unavailable |
| Accent Orange | `#D45F12` | Warnings, pending |

## Breakpoints
| Breakpoint | Width | Target |
|------------|-------|--------|
| xs | < 640px | Mobile |
| sm | 640px | Large mobile |
| md | 768px | Tablet |
| lg | 1024px | Laptop |
| xl | 1280px | Desktop |
| 2xl | 1536px | Large desktop |

## Spacing System
- xs: 0.25rem (4px)
- sm: 0.5rem (8px)
- md: 1rem (16px)
- lg: 1.5rem (24px)
- xl: 2rem (32px)
- 2xl: 3rem (48px)
- 3xl: 4rem (64px)
